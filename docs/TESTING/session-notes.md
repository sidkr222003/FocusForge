# Session Notes Testing

## Manual Test Cases
1. Start and end a session.
2. Confirm the note composer appears when the dashboard is visible.
3. Enter a summary, choose a mood, add blockers, and save.
4. Open History and verify the note appears in the detail panel.
5. Open Journal and confirm the note appears with the correct mood filter.
6. Run `Session Tracker: Add Session Note` and confirm it targets the latest session.

## Expected Results
- Notes are saved to the session record.
- Mood filter hides or shows notes correctly.
- Journal list only shows the last 7 days.
