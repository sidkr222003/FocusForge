# Focus Score Testing

## Manual Test Cases
1. Start a session and remain active for at least 5 minutes.
2. Trigger a break reminder and accept it to reset the streak.
3. Verify the Focus Score gauge updates as active time grows.
4. Open History and confirm the session has a focus score value.
5. Disable focus score in settings and confirm the gauge shows disabled state.

## Expected Results
- Score ranges from 0 to 100.
- Lower idle counts and regular breaks improve the score.
- Disabling focus score removes tooltip display and sets gauge to disabled.
